AngularJS Tutorial

Made by Kurt

AngularJS is a JavaScript framework that can be integrated directly into an HTML page to make it more dynamic. HTML pages, as they stand alone, are static. There are several other avenue's one can take to make static HTML elements more versatile but they involve manipulating the DOM externally (jQuery is a popular example).

"AngularJS lets you extend HTML vocabulary for your application." - AngularJS website. This is done using directives. Directives tag HTML elements with the prefix "ng".
